# Verkefni 5 — Next.js og Sanity CMS

Vefforritun 2, 2026.

## Um verkefnið

Next.js vefur sem sækir efni úr Sanity CMS. Hægt er að breyta öllu efni á síðunni í gegnum Sanity Studio án þess að snerta kóðann.

## Síður

- `/` — Forsíða
- `/posts` — Listi af greinum
- `/posts/[slug]` — Stakur grein
- `/studio` — Sanity Studio

## Keyrsla

```bash
npm install
npm run dev
```

## Tækni

- Next.js með TypeScript og App Router
- Sass fyrir CSS
- Sanity CMS
- Hýst á Vercel
